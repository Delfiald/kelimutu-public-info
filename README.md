# Integrated Website of Kelimutu National Park

## Descriptions

An interactive and informative website for Kelimutu National Park, providing visitors with information on visiting hours, fees, facilities, and guidance.

## Features

- Dual language support (English & Indonesian)
- Sections for history, folklore, and geological formation
- Color changes section for Kelimutu crater lakes
- Interactive gallery showcasing the crater lakes, wildlife, and cultural events
- FAQ section addressing common visitor questions
- Visitor information including opening hours, fees, facilities, and rules
- Cost calculator for estimating visit expenses
- QRIS step-by-step guide for digital payments (informational only)
- Responsive design for mobile and desktop

## Technologies Used

- React
- Vite
- i18next (for internationalization)
- react-pdf (for pdf file reader)

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install the dependencies: `npm install`
